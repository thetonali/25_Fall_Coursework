## 1 实验概述

![image-20251225174135688](E:\Courses\25Fall\AI\CourseCode\assets\image-20251225174135688.png)

本实验使用**糖尿病数据集**（diabetes dataset），包含442个样本和10个特征（年龄、性别、BMI、血压、6项血清指标），目标是预测糖尿病疾病进展的量化指标。实验对比了四种回归方法的性能和特征选择效果。

## 2 模型性能对比分析

### 2.1  **均方误差（MSE）对比**

![image-20251225172845430](E:\Courses\25Fall\AI\CourseCode\assets\image-20251225172845430.png)

| 模型                   | MSE     | 性能提升 |
| ---------------------- | ------- | -------- |
| 简单线性回归（仅BMI）  | 3906.92 | 基准     |
| 多元线性回归（10特征） | 3000.39 | ↓23.2%   |
| Ridge回归（α=0.04）    | 2997.20 | ↓23.3%   |
| Lasso回归（α=0.06）    | 2987.43 | ↓23.5%   |

**分析：**

- 多特征模型比单特征模型性能提升23%以上；
- 正则化方法（Ridge/Lasso）比无正则化略优；
- Lasso表现最佳（MSE=2987.43），但三种多元模型差异很小。

### 2. 2 **超参数选择**

- Ridge最佳α = 0.04：较小的惩罚，说明数据不需要强正则化；
- Lasso最佳α = 0.06：略大于Ridge，体现了其更激进的特征选择策略。

## 4 结果解释

### 4.1  **Lasso的特征选择效果（最重要）**

![image-20251225174400581](E:\Courses\25Fall\AI\CourseCode\assets\image-20251225174400581.png)

Lasso成功剔除了3个特征（age, s2, s4），保留7个核心特征：

**被剔除的特征：**

- **age（年龄）**：系数仅-10.01，对预测贡献极小；
- **s2（某血清指标）**：无正则化时系数476.74，但Ridge压缩到-19.89，说明存在过拟合；
- **s4(某血清指标)**：无正则化时177.06，Ridge压缩到113.95，贡献不稳定。

**解释：** 糖尿病进展主要由代谢指标（BMI、血清指标）和血压决定，年龄不是决定性因素。

### 4.2  **过拟合与正则化效果**

**s1特征的过拟合案例：**

![image-20251225174446079](E:\Courses\25Fall\AI\CourseCode\assets\image-20251225174446079.png)

- 无正则化：-792.18（绝对值最大）；
- Ridge：-161.53（压缩79.6%）；
- Lasso：-90.07（压缩88.6%）。

**分析：** 无正则化模型给s1分配了过大权重，正则化有效控制了这种过拟合倾向。

### 4.3  **多重共线性问题**

**s3特征的符号反转：**

![image-20251225174435994](E:\Courses\25Fall\AI\CourseCode\assets\image-20251225174435994.png)

- 无正则化：+101.04；
- Ridge：-166.68；
- Lasso：-220.21。

**解释：** 符号改变说明s3与其他特征存在**强相关性**（多重共线性）。正则化通过约束系数揭示了变量间的复杂关系。

### 4.4 **核心预测因子**

根据系数大小和稳定性，糖尿病进展的**关键预测因子**为：

![image-20251225174556605](E:\Courses\25Fall\AI\CourseCode\assets\image-20251225174556605.png)

1. **BMI（体重指数）** - 系数~520，三个模型高度一致；
2. **s5（血清指标5）** - 系数~500-750，最大系数；
3. **bp（血压）** - 系数~290-320，稳定的重要因子。

这三个特征在所有模型中系数都很大且符号稳定，是最可靠的预测变量。

## 6 模型选择分析

### (1)推荐Lasso回归

**理由：**

1. **最佳预测性能**：MSE=2987.43（最低）；
2. **自动特征选择**：剔除3个不重要特征，模型更简洁；
3. **可解释性强**：保留的7个特征都有明确的预测价值；
4. **避免过拟合**：有效控制了s1等特征的过大系数。

### (2)Ridge回归的价值：

- 适合所有特征都重要的场景;
- 系数更稳定，不会完全剔除任何特征;
- 如果需要综合考虑所有指标，Ridge是好选择。

## 7 实验结论

1. **多特征建模必要性**：使用全部10个特征比单一BMI特征性能提升23%；
2. **关键预测因子**：
   - **BMI**：肥胖是糖尿病进展的最强指标；
   - **s5**：某种血清指标；
   - **血压**：心血管指标也很重要。
3. **年龄不重要**：年龄对疾病进展预测贡献极小。
4. **正则化价值**：
   - 防止过拟合（s1从-792→-90）；
   - 揭示多重共线性（s3符号反转）；
   - 自动特征选择（Lasso剔除3个特征）。

------

本实验成功展示了正则化方法在实际医疗数据中的应用价值，特别是Lasso在特征选择和模型简化方面的优势。
