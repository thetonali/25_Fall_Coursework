### 4.2 详细分析

#### 4.2.1 线性模型组

**普通线性回归、Ridge、Lasso**：
- 三者性能相对近似，说明：
  - 数据集规模适中，不存在严重的过拟合;
  - 正则化效果不明显;
  - 特征之间的多重共线性问题不严重;
- RMSE=4.9左右，即预测误差约4,900美元;
- MAPE=17%，平均预测误差约为实际房价的17%。

**多项式回归（2次）**：

- R²提升至0.8056，相比线性回归提升14%;
- RMSE降至3.776，误差减少部分;
- 说明特征与房价之间确实存在非线性关系;
- 通过特征交互和高次项捕获了更多信息。

#### 4.2.2 非线性模型组

**随机森林回归**：

- R² = 0.8938，性能优秀;
- 通过集成多棵决策树，自动捕获特征间的复杂交互;
- 对异常值具有较强的鲁棒性;
- 可解释性较强（可提供特征重要性）。

**梯度提升回归**：
- **最佳模型**：R² = 0.9290，RMSE = 2.282.
- 平均预测误差仅为2,282美元;
- MAPE = 9.675%，预测精度最高;
- 通过迭代优化残差，逐步提升预测性能。

**支持向量回归（RBF核）**：
- R² = 0.8262，性能良好;
- RBF核函数能够捕获非线性关系;
- 对参数（C、gamma）较为敏感;
- 计算成本相对较高。

## 6 线性与非线性模型对比

### 6.1 性能差异

| 指标 | 最佳线性模型 | 最佳非线性模型 | 性能提升 |
|------|-------------|---------------|---------|
| R² | 0.8056 | 0.9290 | +5.88% |
| RMSE | 3.776 | 2.282 | -39.57% |
| MAE | 2.575 | 1.778 | -30.95% |
| MAPE | 13.69% | 9.68% | -29.29% |

**关键发现**：

- 非线性模型在所有指标上都显著优于线性模型;
- 误差减少了约31%，说明数据中存在重要的非线性关系;
- 梯度提升回归通过捕获复杂的特征交互实现了最佳性能;

### 6.2 模型复杂度与性能的权衡

```bash
模型复杂度：
Linear < Polynomial < Random Forest < Gradient Boosting < SVR

预测性能：
Linear < Polynomial < SVR < Random Forest < Gradient Boosting

可解释性：
Linear > Polynomial > Random Forest > Gradient Boosting > SVR
```

## 7 实验结论

1. LSTAT（低收入人口比例）是最重要的特征，RM（房间数）是第二重要的特征。这两个特征贡献了约81.72%的预测能力。
   
2. 多项式回归比线性回归提升14%，非线性模型(梯度提升)比线性模型误差减少，说明非线性关系普遍存在。
   
3. 各个模型的测试效果中，梯度提升回归表现最佳（R²=0.9290），随机森林次之（R²=0.8938），但可解释性更好。
   
4. 多个特征的组合比单个特征效果更好，非线性模型能够自动捕获特征交互。
