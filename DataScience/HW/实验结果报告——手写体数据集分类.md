# 实验结果报告——手写体数据集分类

## 1 概述

### 1.1 作业要求

1. 对比SVM和CNN两种分类器在手写体数据集上的分类性能；
2. 评估分类器在不同数据集上的表现差异；
3. 分析CNN模型的跨数据集迁移能力。

### 1.2 数据集说明

1. **MNIST**: 手写数字数据集，包含0-9共10类数字，训练集60,000张，测试集10,000张；
2. **Fashion-MNIST**: 服装图像数据集，包含10类服装，训练集60,000张，测试集10,000张；
3. **EMNIST-Digits**: 扩展手写数字数据集，与MNIST类似但来源更广泛。

------

## 2 模型

### 2.1 SVM分类器

- **核函数**: RBF (径向基函数)；
- **正则化参数**: C = 5；
- **Gamma参数**: scale (自动缩放)；
- **数据预处理**: StandardScaler标准化；
- **训练样本**: 为加快训练速度，采样10,000个样本；

SVM通过寻找最优超平面来分类数据，RBF核能够处理非线性问题。标准化预处理对SVM性能至关重要，可以确保不同特征维度具有相同的尺度。

### 2.2 CNN分类器

```
输入层: (28, 28, 1)
├─ 卷积块1: 2×Conv2D(32) + BatchNorm + MaxPool + Dropout(0.25)
├─ 卷积块2: 2×Conv2D(64) + BatchNorm + MaxPool + Dropout(0.25)
├─ 卷积块3: Conv2D(128) + BatchNorm + Dropout(0.25)
├─ 展平层: Flatten
├─ 全连接层1: Dense(256) + BatchNorm + Dropout(0.5)
├─ 全连接层2: Dense(128) + Dropout(0.5)
└─ 输出层: Dense(10, softmax)
```

## 5 跨数据集迁移学习验

**跨领域迁移**: MNIST → Fashion-MNIST (数字 vs 服装)。

验证在MNIST上训练的CNN模型能否迁移到其他数据集，评估特征的可迁移性。

### 5.1 跨领域迁移结果 (MNIST → Fashion-MNIST)

![Figure_21](C:\Users\Slavo\Desktop\worker\Figure_21.png)

| 测试方式                      | 准确率  | 分析              |
| ----------------------------- | ------- | ----------------- |
| **直接迁移**                  | 10.58%  | 接近随机猜测(10%) |
| **在Fashion-MNIST上独立训练** | 93.26%  | 基准性能          |
| **性能差距**                  | -82.68% | **迁移完全失败**  |

**失败原因——**

1.特征完全不匹配。
    MNIST学习的是笔画、曲线、角度等数字特征；
    Fashion-MNIST需要纹理、形状、轮廓等服装特征；
    两者特征空间几乎不重叠。

2.低层特征差异。
    数字图像: 黑色笔画 + 白色背景；
    服装图像: 复杂纹理 + 灰度变化；
    边缘检测器学到的模式完全不同。

**可视化说明——**
    10.58%的准确率说明模型几乎在随机猜测；
    混淆矩阵应该显示分布接近均匀(每类约10%)；
    证明了深度学习模型的特征学习具有高度任务特异性。

**结论——**
    跨领域迁移在没有微调的情况下几乎无效；
    迁移学习的成功高度依赖于源域和目标域的相似性。

## 6 综合对比分析

### 6.1 SVM vs CNN 全面对比

| 指标                | SVM    | CNN      | 胜者           |
| ------------------- | ------ | -------- | -------------- |
| **MNIST准确率**     | 95.13% | 99.53%   | CNN (+4.40%)   |
| **Fashion准确率**   | 86.66% | 93.26%   | CNN (+6.60%)   |
| **MNIST训练时间**   | 6.46s  | 1075.28s | SVM (快166倍)  |
| **Fashion训练时间** | 5.64s  | 1107.22s | SVM (快196倍)  |
| **MNIST测试时间**   | 16.23s | 8.31s    | CNN (快1.95倍) |
| **Fashion测试时间** | 19.13s | 7.97s    | CNN (快2.40倍) |

**SVM性能下降**:
MNIST → Fashion-MNIST: 95.13% → 86.66% (**-8.47%**)；
相对下降: 8.9%；
原因: RBF核难以捕捉复杂纹理特征。

**CNN性能下降**:
MNIST → Fashion-MNIST: 99.53% → 93.26% (**-6.27%**)；
相对下降: 6.3%；
原因: 服装类别间相似度高，需要更精细的特征。

**观察发现**:
CNN对数据复杂度的鲁棒性更强(相对下降6.3% vs 8.9%)；
在简单任务上两者差距小(4.40%)；
在复杂任务上差距扩大(6.60%)。

### 6.2 训练效率与性能权衡

**发现**:
SVM: 高效但性能受限于95%左右的上限；
CNN: 需要大量训练时间，但可突破性能瓶颈；
选择策略——
    准确率要求<90%: 优先选择SVM；
    准确率要求>95%: 必须使用CNN；
    介于90-95%之间: 权衡开发周期和性能需求。

### 6.4 不同类别的识别难度

#### MNIST数字识别难度排序

**易识别** (F1 > 0.99):
数字"0" - 闭合圆形，特征明显数字；
"1" - 简单直线，最易区分数字；
"6" - 独特形状。

**中等难度** (0.94 < F1 < 0.99): 
数字"2,3,4,5,8,9" - 有一定混淆可能。

**困难识别** (F1 < 0.94): 
数字"7" - 与"1"相似，书写风格差异大。

#### Fashion-MNIST服装识别难度排序

**易识别** (F1 > 0.98):
Trouser(99.25%) - 裤子形状独特；
Bag (98.95%) - 包的轮廓清晰；
Sandal(99.05%) - 凉鞋特征明显。

**中等难度** (0.90 < F1 < 0.98): 
Sneaker(97.06%) - 运动鞋特征较清晰； 
Ankle boot(97.20%) - 靴子与鞋略有混淆 ；
Dress (93.00%) - 裙子形状可辨。

**困难识别** (F1 < 0.90): 
Pullover (90.20%) - 套头衫与外套相似 ；
Coat (89.70%) - 外套与套头衫混淆；
T-shirt (89.06%) - 与衬衫相似；
Shirt (78.87%) - **最难**，与T-shirt/Pullover高度相似。

**难度分析**:

- 下半身服装(Trouser, Ankle boot) > 配饰(Bag, Sandal) > 上半身服装；
- 上衣类别(Shirt, T-shirt, Pullover, Coat)相互混淆严重；

   相似领域迁移(MNIST→EMNIST)有效: 保留90%+性能；

   迁移成功的关键—— 源域和目标域的高度相似性。
